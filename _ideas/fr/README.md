# Idées en Français

Placez les descriptions d'idées en français dans ce répertoire.

Les fichiers doivent être nommés : `nom-idee.md`

Chaque idée doit inclure le front matter :
```yaml
---
layout: idea
title: "Titre de l'idée"
lang: fr
date: 2026-01-22
tags: [tag1, tag2]
stage: elaborated
---
```
