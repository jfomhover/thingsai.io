# ThingsAI

Ideas, learning, and explorations in AI and computational thinking by Jeff Omhover.

## Development

This site is built with Jekyll 4.3.0.

### Local Setup

```bash
bundle install
bundle exec jekyll serve --port 4001
```

Visit: http://localhost:4001

### Site Structure

- `/en/blog/` - Blog posts
- `/en/memos/` - Quick reference memos
- `/en/ideas/` - Idea showcase
- `/en/essays/` - Long-form essays
- `/fr/` - French versions (blog, mémos, idées)

### Collections

- `_posts` - Blog posts
- `_flashcards` - Quick memos/flashcards
- `_ideas` - Ideas and concepts
- `_essays` - Long-form writing
