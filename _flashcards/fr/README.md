# Mémos en Français

Placez les résumés en français dans ce répertoire.

Les mémos sont des résumés concis d'une page sur les meilleures pratiques ou les connaissances clés sur un sujet spécifique. Ils distillent le matériel de `context/knowledge/` en pages de référence réutilisables et bien formatées.

Les fichiers doivent être nommés: `nom-du-sujet.md`

Chaque mémo doit inclure un en-tête:
```yaml
---
layout: flashcard
title: "Titre du Mémo"
subtitle: "Sous-titre optionnel pour le contexte"
lang: fr
date: 2026-01-22
topic: nom-du-sujet  # ex: redaction-cv, agents-ia, carriere
tags: [tag1, tag2, tag3]
source: "context/knowledge/best-practices/fichier-source.md"  # Optionnel
---
```

## Lignes directrices

- **Une page**: Gardez le contenu concis et axé sur les points clés
- **Signal élevé**: Incluez uniquement des informations essentielles et exploitables
- **Bien formaté**: Utilisez des titres clairs, des points de balle et une hiérarchie visuelle
- **Facile à lier**: Structure pour faciliter les liens croisés depuis les articles de blog et les essais
- **Qualité référence**: Le contenu doit servir de guide de référence rapide
