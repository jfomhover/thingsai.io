# Essais en Français

Placez les essais (articles de fond) en français dans ce répertoire.

Les fichiers doivent être nommés : `YYYY-MM-DD-titre-essai.md`

Chaque essai doit inclure le front matter :
```yaml
---
layout: essay
title: "Titre de l'essai"
lang: fr
date: 2026-01-22
categories: [categorie1, categorie2]
---
```
