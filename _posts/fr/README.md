# Articles de Blog en Français

Placez les articles de blog en français dans ce répertoire.

Les fichiers doivent être nommés : `YYYY-MM-DD-titre-article.md`

Exemple :
```
2026-01-22-mon-premier-article.md
```

Chaque article doit inclure le front matter :
```yaml
---
layout: post
title: "Titre de l'article"
lang: fr
date: 2026-01-22
categories: [categorie1, categorie2]
---
```
