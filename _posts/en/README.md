# English Blog Posts

Place English blog posts in this directory.

Files should be named: `YYYY-MM-DD-post-title.md`

Example:
```
2026-01-22-my-first-post.md
```

Each post should include front matter:
```yaml
---
layout: post
title: "Post Title"
lang: en
date: 2026-01-22
categories: [category1, category2]
---
```
